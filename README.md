# CMS
This is a web application i have built using gemini pro. The purpose of the application is to create custom cold emails to professors and send it to them from a single web page. No continous copy paste, multiple mails, editing, etc.
</br>
</br>
It can be used by anyone just by providing their Gemini api and Gmail Client ID
